[minilisp](https://github.com/rui314/minilisp)
用 1000 行 C 语言写的 Lisp 解释器。实现了整数、符号、局部变量、条件语句、宏和垃圾回收等功能，为了让读者更好地理解项目，代码中包含了丰富的注释
```
(println 3)  ; prints "3"
(+ 1 2 3)  ; -> 6
(define a (+ 1 2))
(+ a a)  ; -> 6
```
